Intelligent Group Movement Prototype

Movement based AI for large numbers of simultaneously interacting game objects

What it is:

This prototype was created as a way to provide emergent movement elements for a game.

In the style of a Real-Time Strategy game, the Army Marching prototype features a series troop formations that can be edited to create various geometric formations. These formations can changed on the fly resulting in the troops taking intelligent paths to their new adjusted locations. The result creates very fun and useful ways of moving small to large groups of objects which can be implemented in many genres.

Engine: Unity
Platform: PC, Mac, WebGL

Square Formation Movements

Control over each row with live changes to spacing

Ring Formation Movements

Concentric ring formations shown with automatic adjustments per unit
